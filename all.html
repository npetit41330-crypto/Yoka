Privacy Policy

Nomy respecte votre vie privée. Cette politique décrit quelles données sont collectées, pourquoi elles le sont et comment elles sont protégées.

Données collectées :

  • ID Discord : identifiant unique de chaque utilisateur.
  • ID serveur et salons : nécessaires pour les commandes, logs et tickets.
  • Messages : utilisés uniquement pour la modération et les tickets.
  • Rôles et niveaux : pour le système d’XP et l’attribution automatique des rôles.
  • Logs de modération : avertissements, mutes, bans, fermetures de tickets.
  • Transcriptions de tickets : sauvegardées pour le suivi du support et du serveur.

Utilisation des données :

  • Assurer le fonctionnement des fonctionnalités du bot.
  • Gérer les niveaux et rôles automatiques.
  • Fournir un support via tickets privés.
  • Enregistrer les logs pour sécurité et transparence.

Partage et sécurité :

  • Aucune donnée n’est partagée ou vendue à des tiers.
  • Les informations sont accessibles uniquement aux membres du staff autorisés.
  • Les données sont conservées uniquement pour le bon fonctionnement du bot.

Suppression des données :

  • Les données peuvent être supprimées si le bot est retiré du serveur.
  • Demande de suppression possible via un administrateur du serveur.

Privacy Policy

Nomy values your privacy. This policy explains what data is collected, why, and how it is protected.

Data Collected:

  • Discord ID: unique identifier for each user.
  • Server and channel IDs: required for commands, logs, and tickets.
  • Messages: used only for moderation and support tickets.
  • Roles and levels: for XP and automatic role assignment.
  • Moderation logs: warnings, mutes, bans, ticket closures.
  • Ticket transcripts: saved for support tracking.

Data Usage:

  • Ensure bot commands and features work properly.
  • Manage XP and automatic roles.
  • Provide private support via tickets.
  • Record logs for security and transparency.

Sharing and Security:

  • No data is sold or shared with third parties.
  • Data is accessible only to authorized staff members.
  • Data is kept only as long as necessary for bot functionality.

Data Deletion:

  • Data can be deleted if the bot is removed from the server.
  • You may request deletion via a server administrator.
